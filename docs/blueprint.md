# **App Name**: AlquilaTeam

## Core Features:

- Centralized Listing: Allow users to add and centralize rental listings from various sources by simply pasting the link, which will create a shareable listing for everyone in the group.
- Cost Calculator: Calculate the total cost of rentals by adding rent and other expenses.
- Adjustment Tracking: Track adjustment types (ICL or IPC) and frequency to anticipate rent increases. This involves storing adjustment data and calculating future rent.
- Voting System: Enable users to vote on favorite properties.
- Real-time Collaboration: Enable multiple users to view and interact with the rental listings and discussions in real-time, fostering teamwork.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to inspire trust and organization.
- Background color: Very light blue (#F0F8FF), almost white, for a clean and transparent look.
- Accent color: Soft orange (#FFB347) for call-to-action buttons and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, suitable for headlines or body text.
- Simple and clean icons for listing features, voting actions and alerts.
- Minimalist layout with a focus on clear data presentation in a shared dashboard format.
